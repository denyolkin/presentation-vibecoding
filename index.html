Failures & cost

Voice-to-text → hallucination chain

Wispr Flow misrecognized names. Wrong names hit the prompt → AI hallucinated on top. Caught it before it went out.

AI deleted its own API keys

Cursor overwrote .env while editing config. System went dark. Lesson: .env is now read-only.

$$$: Runaway API calls

System decided to parse my personal inbox via API. Found out when credit card charge notifications started flooding my phone.

Cheat codes

claude -p on a server

Claude Max ($100-200/mo) includes unlimited CLI use. Run claude -p "prompt" on your server instead of paying per API token. Saves thousands.
